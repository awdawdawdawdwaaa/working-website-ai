# Ved - Scroll Cinematic Flow

A production-ready Vite, React, and Three.js portfolio experience built as one continuous scroll-scrubbed 3D journey.

## Flow

1. Intro: character in a dark void with a focused overhead spotlight.
2. Ahmedabad map: illuminated wall map with location text.
3. Door transition: hinged door opens as the character approaches.
4. Room reveal: dark workstation room lit by a desk lamp.
5. Sit down: character aligns to the chair and settles into the workstation.
6. Monitor on: project UI appears only on the monitor surface.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The finished static site is emitted to `dist/`.
